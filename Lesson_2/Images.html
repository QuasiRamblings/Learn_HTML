This is how to use HTML for images

Insert images into HTML using the img tag with a source attribute src

Best practice is to always include 'alt' tag within element - it's used to describe the image for the visually impaired for screen readers

Important to know how big your image is to be able to add attributes for height and width in text of pixels, i.e. 500x375

Open the image, then right click on it, select file info, and look for the dimensions

Block-level elements will take up the entire width of the page

In-line elements will only take up as much space as it's widdth

Paragraph tags are Block-level elements and take up the entire width, so nothing will go next to it
