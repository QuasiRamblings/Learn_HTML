(3) Types of Lists

Unordered Lists - Look like bulle points lists

Uses the "ul" tag and then "li" for list item

  • Carrots
  • Cucumbers
  • Tomatoes

Order Lists - which are numbered lists

Uses the "ol" tag

  1. Carrots
  2. Cucumbers
  3. Tomatoes

Descriptive Lists - which have a parent child look to them - gives a description of the title of the item

Uses the "dl" tag

Carrots
Orange
Cucumbers
Green
Tomatoes
Red
